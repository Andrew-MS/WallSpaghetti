# WallSpaghetti

Monorepo for WallSpaghetti services and apps.

## Structure

- `services/` — backend services (APIs, workers)
- `apps/` — frontends (web, mobile)
- `packages/` — shared libraries
- `pocs/` — proof of concept experiments
- `infra/` — infrastructure as code
- `docs/` — architecture decisions, notes

## Getting Started

Each service/app has its own README with setup instructions.
